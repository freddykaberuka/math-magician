# Math Magicians

"Math magicians" is a website solve mathematics calculation.

### Built With

 - React,
 - css
 - Linters

## Getting Started

### Setup

Clone or Download Repository
Extract the download Folder

### Install

after cloning Repository [HERE]([url](https://github.com/freddykaberuka/math_magician/) open the folder and run npm install to install all the modules in your local storage

### Usage

Open the extracted folder and run 
`npm install`
`npm start` to start a server

and for the contributor i recommend to use [VSCode](https://code.visualstudio.com/) as text editor and Live Server installed on it.
